Choosing Your Tool

Which Claude Model Should You Use?

Different tasks in the fund require different Claude interfaces. Here's how to choose — and the exact prompt structure for each.

🖥️ Cowork PRIMARY

Best for: Building files (Excel models, Word LP reports, PowerPoint decks), running multi-step workflows, automating repetitive fund tasks.

Use Cowork when you need Claude to actually CREATE something — a financial model, a quarterly report, a DD checklist.
✓ Creates .xlsx, .docx, .pptx ✓ Multi-step tasks ✓ Runs code
💬 Claude Chat

Best for: Research, analysis, drafting investment memos, answering LP questions, reviewing documents, thinking through deal thesis.

Use Chat for quick Q&A, brainstorming, and when you need analysis or writing — not file creation.
✓ Research ✓ Analysis ✓ Drafting
⌨️ Claude Code

Best for: Building automation scripts, Power Automate flows, Python data pipelines, Excel macros, API integrations.

Use Claude Code when you're building the infrastructure — the scripts and flows that run the AI agents automatically.
✓ Python scripts ✓ Automation ✓ APIs

🎯 The Golden Rule for This Demo

Use Cowork as your default. Switch to Claude Code only when building the underlying automation infrastructure. Use Chat for any quick analysis during a live demo.

Quarterly management fee accrual: 2% of $50M = $1M/year ÷ 4 = $250,000.
